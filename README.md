## Overview

A browser-based Minecraft experience designed for players who can't afford the official game or use Chromebooks. Play Minecraft with built-in cheats and features directly from your web browser - no installation required.

## ✨ Features

- 🌐 **Browser-Based**: Play directly in your web browser
- 💾 **No Installation**: Works on Chromebooks and low-spec devices
- ⚡ **Cheats Enabled**: Access to game-enhancing modules
- 🚀 **Lightweight**: Optimized for accessibility
- 🎯 **Free to Play**: No cost, no paywall

## 🚀 Getting Started
